Worklight Dynamic Busy Indicator Tests



This tests out the dynamic busy indicator.

In hybrid applications, the native Worklight busy indicator is used. For browser based applications, the Dojo Mobile busy indicator emulation is used. IOS devices allow for a custom message to be supplied. For all others, the custom message is ignored. Once started the busy indicator will run for 5 seconds, and then be closed.

Syntax:

require(["issw/mobile/activity"], function(activity) {
    activity.start("Optional message");
    //-- Do something
    activity.stop();
})
